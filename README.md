# AntiRUB Governance Repository

## азначение репозитория

тот репозиторий является **единым публичным пространством управления (governance)**
сообщества AntiRUB на ранней стадии развития.

десь:
- формулируются и фиксируются принципы сообщества;
- выдвигаются и обсуждаются инициативы;
- принимаются и документируются решения;
- сохраняется неизменяемая история обсуждений и договорённостей.

епозиторий создан для обеспечения **прозрачности, ответственности и защиты от произвольных изменений правил**.

---

## лючевой принцип

> **нициатива  обсуждение  решение  фиксация**

икакие решения не принимаются неформально или по умолчанию.

---

## Структура репозитория (кратко)

- principles/  базовые ценности и принципы AntiRUB
- egulations/  регламенты и правила взаимодействия
- proposals/  инициативы и предложения (RFC)
- decisions/  принятые решения
- ledger/  связка решений с on-chain фиксацией
- 	emplates/  шаблоны документов
- discussions/  ссылки на ключевые обсуждения

---

## ак предложить инициативу

1. знакомьтесь с принципами сообщества (principles/) и правилами обсуждений (egulations/discussion-rules.md).
2. Создайте RFC по шаблону 	emplates/proposal-template.md и сохраните его в proposals/ с уникальным номером.
3. ткройте обсуждение через GitHub Issues или Discussions и добавьте ссылку на RFC.
4. удьте готовы участвовать в реализации.

нициативы без личного вклада не рассматриваются.

---

## Статус

епозиторий находится в режиме **ранней стадии (520 участников)**.
роцедуры усложняются только по мере роста сообщества и уровня рисков.

> **сли решение нельзя проверить, перечитать и воспроизвести  его не существует.**
